# 🐍 Python Algorithm Journey

This repository tracks my learning progress in Python programming, focusing on **Algorithm Logic**, **Data Structures**, and **Automation (RPA)**. All projects are developed within a **WSL (Ubuntu)** environment.

---

## 📖 Course Curriculum
Based on the "Programação de Algoritmos em Python" official guide:
- **Fundamentals:** Variables, dynamic typing, and I/O operations.
- **Logic & Control:** Boolean operators and decision-making structures (`if/else`).
- **Iterations:** Loop automation using `for` and `while`.
- **Data Collections:** Managing information with Lists and Tuples.
- **Modularization:** Function development and Exception Handling (`try/except`).

---

## 🛠️ Tech Stack & Environment
* **OS:** Ubuntu Linux (via WSL)
* **Language:** Python 3.12+
* **Development:** Terminal-based (Nvim/Vim)
* **Environment Management:** Python `venv` (Virtual Environments)
* **Version Control:** Git with SSH authentication

---

## 🧠 Applied Concepts
- **Clean Code:** Variables and documentation written in English.
- **Cross-platform Compatibility:** Using `pathlib` for robust file path management.
- **Professional Workflow:** Semantic commits and organized repository architecture.
